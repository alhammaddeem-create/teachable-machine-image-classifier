# teachable-machine-image-classifier
Image recognition project using Google Teachable Machine to classify Strawberry and Cherry images. The project includes a TensorFlow Keras model, Python prediction script, evaluation results, and documentation.

## Overview

This project demonstrates an image recognition model created using **Google Teachable Machine**. The model classifies images into two categories:

* Strawberry 
* Cherry 

The trained model was exported in **TensorFlow → Keras** format and used in a Python script to predict the class of an input image.

---

# Classes

* Strawberry
* Cherry

---

# Tools Used

* Google Teachable Machine

* Python

---

# Project Files

* `predict.py` – Python script for image classification.
* `keras_model.h5` – Trained Keras model.
* `labels.txt` – Class labels.
* `test_images/` – Sample images used for testing.
* `screenshots/` – Screenshot showing the prediction result.

---

# How to Run

1. Install the required libraries.
2. Place the model files in the `model` folder.
3. Run the Python script.
4. Select an input image.
5. The program predicts the image class.

---

# Results

The trained model successfully classified Strawberry and Cherry images with high confidence using the exported TensorFlow Keras model.
